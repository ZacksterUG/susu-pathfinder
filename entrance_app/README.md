# Разметка входов в аудитории СУСУ

Приложение для установки точек входа в аудитории и лестницы корпусов университета.

## 🚀 Запуск

```bash
python main.py
```

## 🕸️ Генерация сетки коридоров

1. **Выберите размер клетки** слайдером (5-100 пикселей)
2. **Нажмите "🕸️ Построить сетку"** — узлы генерируются внутри коридора, исключая аудитории/лестницы/лифты
   - Узлы на контуре препятствий автоматически удаляются
3. **"👁️ Показать/скрыть"** — переключение видимости сетки
4. **"🔗 Подключить входы"** — точки входа соединяются перпендикулярно с ближайшими рёбрами
5. **"💾 Сохранить сетку"** — сохранение в `grid.json`

**Особенности:**
- Узлы, лежащие на контуре аудиторий/лестниц/лифтов/тех.помещений, **автоматически удаляются**
- При подключении входа:
  - Находится ближайшее ребро сетки
  - Добавляется новый узел в точке перпендикулярной проекции
  - Добавляются рёбра от нового узла к концам исходного ребра
  - Добавляется ребро от точки входа к новому узлу
- **Технические помещения** (туалеты, охрана, подсобки, гардеробы, кафетерии) отображаются белым, входы для них запрещены

**Формат grid.json:**
```json
{
  "9c7a0f84-03b0-40b0-8fb0-992e1008c11e_1": {
    "cell_size": 20,
    "nodes": [
      {"x": 100, "y": 150},
      {"x": 120, "y": 150},
      ...
    ],
    "edges": [
      {"from": 0, "to": 1, "weight": 20},
      ...
    ],
    "entrance_connections": [
      {
        "entrance_id": "room-123",
        "entrance_x": 150,
        "entrance_y": 200,
        "connection_node_idx": 45
      }
    ],
    "building_name": "Западное крыло главного корпуса",
    "floor": "1"
  }
}
```


## 📋 Как пользоваться

1. **Запустите приложение:** `python main.py`
2. **Выберите корпус** из выпадающего списка (отсортированы по алфавиту)
3. **Выберите этаж** — карта загрузится автоматически
4. **Наведите на аудиторию** (синяя) или **лестницу** (красная)
   - Белая точка с красной обводкой появится на контуре здания
   - Координаты отобразятся внизу окна
5. **Кликните** для установки точки входа
6. **Сохраните** через кнопку "💾 Сохранить входы"

## 🎨 Обозначения на карте

| Цвет | Объект |
|------|--------|
|  Серый | Коридор |
| 🔵 Синий | Аудитория |
| 🔴 Красный | Лестница |
| 🟡 Жёлтый | Лифт |
|  Белый | Техническое помещение (туалет, охрана, подсобка) |
| 🟢 Зелёный | Точка входа |
| ⚪ Белый с красным | Предпросмотр (на контуре) |
| 🔵 Голубой | Узел сетки |
| 🟢 Тёмно-зелёный | Ребро сетки |
| 🟢 Ярко-зелёный (большой) | Точка подключения входа |

**Важно:** Для технических помещений (белые) **нельзя** устанавливать точки входа.

## 📊 Формат данных entrances.json

```json
{
  "9c7a0f84-03b0-40b0-8fb0-992e1008c11e_1": {
    "room-id-123": {
      "x": 150,
      "y": 200,
      "room_number": "101",
      "type": "room"
    },
    "stair-id-456": {
      "x": 300,
      "y": 250,
      "room_number": "Лестница 1",
      "type": "stairs"
    }
  }
}
```

**Ключи:**
- `{buildingId}_{floor}` — корпус и этаж
- `{objectId}` — ID аудитории или лестницы

**Поля:**
- `x`, `y` — целочисленные координаты точки входа
- `room_number` — номер аудитории/лестницы для отображения
- `type` — тип объекта (`room`, `stairs`, `elevator`)

## ⚙️ Требования

- Python 3.8+
- tkinter (встроен в Python)
- Pillow (опционально)

```bash
pip install -r requirements.txt
```

## 🔧 Особенности

- Точка входа устанавливается **только на контуре** аудитории/лестницы
- Координаты **целочисленные**
- Предпросмотр "примагничивается" к ближайшей точке на контуре
- Список корпусов отсортирован по полному названию
